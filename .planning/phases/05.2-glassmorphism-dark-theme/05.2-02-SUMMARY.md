## Decisions Made
- TaskRow glass cards use `mb-2` to separate rows instead of `border-b` — each row is its own glass surface with hover animation
- ReflectionModal blur overridden to 20px via inline style (vs .glass default 16px) — modal should feel most elevated of all surfaces
- calendar.tsx v9 keys are a non-trivial remap: v8 `head_row`/`head_cell`/`row`/`cell`/`day` silently ignored in v9 causing blank calendar cells; full remap required
- EmptyState and TodayView header both use `mx-4 inset` pattern for consistency across glass card surfaces
