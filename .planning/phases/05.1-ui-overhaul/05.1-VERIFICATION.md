## Notes

**D-25 / EmptyState alignment detail:** EmptyState uses `px-4 py-8` and renders inside TodayView's `<div className="mx-6">` wrapper — the same wrapper that contains TaskRows (also `px-4`). Left-edge alignment between empty state text and task titles is therefore consistent at the component level. The "flush-left" goal is achieved.

**D-18 / Nav bg-card light mode caveat:** In the default shadcn light theme, `--card` and `--background` both resolve to `hsl(0 0% 100%)` — pure white. The visual panel distinction from `bg-card` is therefore imperceptible in light mode. This is a known limitation of the shadcn token defaults, not an implementation error. The decision remains correct and will show effect in dark mode (where `--card: 222.2 84% 4.9%` matches `--background: 222.2 84% 4.9%` — also identical in default dark). The intent is preserved for theme customization or a future dark mode redesign. This is flagged for human awareness but does not constitute a gap.
