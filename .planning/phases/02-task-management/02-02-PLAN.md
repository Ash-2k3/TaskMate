- Entire row clickable: calls `onClick(task.id)`

Row wrapper classes:
- Base: `flex items-center gap-3 px-4 py-4 bg-muted border-b border-border cursor-pointer hover:bg-accent`
- Low priority: add `opacity-60` to the row
- No animations, no transitions

CRITICAL date handling: import `parseISO` and `differenceInCalendarDays` from `date-fns`. To compare dates: `differenceInCalendarDays(new Date(), parseISO(task.due_date))`. Never use `new Date(task.due_date)` directly (UTC timezone bug).
