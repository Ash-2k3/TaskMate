1. **Create `src/renderer/components/DatePicker.tsx`:**

A wrapper around shadcn Popover + Calendar. Props: `value: string | null` (ISO date string "YYYY-MM-DD" or null), `onChange: (date: string | null) => void`.

- Use `Popover`, `PopoverTrigger`, `PopoverContent` from `@/components/ui/popover`
- Use `Calendar` from `@/components/ui/calendar`
- Use `Button` from `@/components/ui/button` as the trigger
- Display: when value is null show "No due date" in muted-foreground. When set, show formatted date like "Fri, Mar 21" using `format(parseISO(value + 'T00:00:00'), 'EEE, MMM d')` from date-fns. CRITICAL: append `T00:00:00` to prevent UTC shift.
- Calendar `selected` prop: `value ? new Date(value + 'T00:00:00') : undefined`
- Calendar `onSelect` handler: when a date is selected, call `onChange(format(selectedDate, 'yyyy-MM-dd'))`. When cleared (if supporting clear), call `onChange(null)`.
- Add a "Clear" text button below calendar inside the popover to reset to null
- Close popover on date select by managing open state with `useState`
