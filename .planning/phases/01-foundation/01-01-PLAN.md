    **forge.config.ts** — Update to include native module handling (per FOUND-01, FOUND-04):
    ```typescript
    import type { ForgeConfig } from '@electron-forge/shared-types';
    import { VitePlugin } from '@electron-forge/plugin-vite';

    const config: ForgeConfig = {
      packagerConfig: {
        asar: true,
        asarUnpack: ['**/*.node', '**/better-sqlite3/**'],
      },
      rebuildConfig: {
        onlyModules: ['better-sqlite3'],
      },
      makers: [
        { name: '@electron-forge/maker-squirrel', config: {} },
        { name: '@electron-forge/maker-zip', platforms: ['darwin'] },
        { name: '@electron-forge/maker-deb', config: {} },
      ],
      plugins: [
        new VitePlugin({
          build: [
            { entry: 'src/main/index.ts', config: 'vite.main.config.ts' },
            { entry: 'src/preload/preload.ts', config: 'vite.preload.config.ts' },
          ],
          renderer: [
            { name: 'main_window', config: 'vite.renderer.config.ts' },
          ],
        }),
      ],
    };
    export default config;
    ```

    **vite.main.config.ts** — Mark native modules as external (per FOUND-04 — better-sqlite3 must not be bundled):
    ```typescript
    import { defineConfig } from 'vite';
    export default defineConfig({
      build: {
        rollupOptions: {
          external: ['better-sqlite3', 'electron-store', 'electron'],
        },
      },
    });
    ```

    **vite.preload.config.ts** — Externalize electron:
    ```typescript
    import { defineConfig } from 'vite';

          external: ['electron'],
        },
      },
    });
    ```

    **vite.renderer.config.ts** — Add React plugin and CRITICAL base './' (per FOUND-01 — prevents white screen):
    ```typescript
    import { defineConfig } from 'vite';
    import react from '@vitejs/plugin-react';
    export default defineConfig({
      base: './',
      plugins: [react()],
    });
    ```

    **src/renderer/index.html** — Add CSP meta tag (per FOUND-05 — offline only, no network access):
    Add inside `<head>`:
    ```html
    <meta http-equiv="Content-Security-Policy"
          content="default-src 'self'; script-src 'self'; style-src 'self' 'unsafe-inline'; img-src 'self' data:;">
    ```
    Ensure there is a `<div id="root"></div>` inside `<body>`.

    **src/renderer/main.tsx** — React entry with HashRouter (per FOUND-01 — no BrowserRouter):
    ```tsx
    import React from 'react';
    import { createRoot } from 'react-dom/client';
    import { HashRouter } from 'react-router-dom';
    import App from './App';

    const root = createRoot(document.getElementById('root')!);
    root.render(
      <React.StrictMode>
        <HashRouter>
          <App />
        </HashRouter>
      </React.StrictMode>
    );
    ```

    **src/renderer/App.tsx** — Minimal App component with health check:
    ```tsx
    import React from 'react';

    function App() {
      const ipcReady = typeof window !== 'undefined' && !!(window as any).taskmate;
      return (
        <div style={{ padding: '2rem', fontFamily: 'system-ui' }}>
          <h1>TaskMate</h1>
          <p>Foundation loaded.</p>
          <p>IPC bridge: {ipcReady ? 'connected' : 'not yet connected'}</p>
        </div>
      );
    }

    export default App;
    ```

    **src/main/index.ts** — BrowserWindow with security baseline and devtools gate (per FOUND-01, FOUND-06):
    Set up the main entry with:
    - `nodeIntegration: false`
    - `contextIsolation: true`
    - preload path using `path.join(__dirname, '../preload/preload.js')` (note: .js not .ts — Vite compiles it)
    - `show: false` + `ready-to-show` event for faster perceived startup (FOUND-06)
    - DevTools gated on `!app.isPackaged`
    - Forge-injected constants: `declare const MAIN_WINDOW_VITE_DEV_SERVER_URL: string;` and `declare const MAIN_WINDOW_VITE_NAME: string;`
    - Dev: `mainWindow.loadURL(MAIN_WINDOW_VITE_DEV_SERVER_URL)`
    - Prod: `mainWindow.loadFile(path.join(__dirname, '../renderer/${MAIN_WINDOW_VITE_NAME}/index.html'))`
    - `app.on('window-all-closed', ...)` — do NOT quit on macOS (noop for now; tray handled in 01-03)
    - Windows AUMID: `if (process.platform === 'win32') app.setAppUserModelId('com.taskmate.app');` — must be called early in `app.whenReady()`
